# Accounts Payable Analysis 2024

## Overview
This project analyzes Accounts Payable (AP) data to identify spending patterns, vendor performance, payment trends, and outstanding balances.

## Objectives
- Analyze invoice activity.
- Identify top vendors by total spending.
- Monitor payment status and outstanding balances.
- Generate insights to support financial decision-making.

## Files Included

| File | Description |
|--------|-------------|
| AP Analysis 2024.xlsx | Excel analysis and dashboard |
| invoices.csv | Invoice transaction data |
| vendors.csv | Vendor information |
| ap_analysis.sql | SQL queries used for data analysis |

## Tools Used
- SQL
- Microsoft Excel
- CSV Data Sources
- GitHub

## Analysis Performed
- Total amount invoiced
- Total amount paid

- Top vendors by spend
